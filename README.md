# virtual-workspaces

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

## About this project

`viwo`, or **vi**rtual **wo**rktrees, is a tool to help scaffold virtualized
environments on your local machine for you to run AI Agents. This brings the following
perks:

- Security - Limit potential damage done by AI agents by limiting their scope to a specific folder.
